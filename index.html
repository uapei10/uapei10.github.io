ABOUT OUR PROJECT

Context

This Project is part of the project PASMO-"Plataforma Aberta para o desenvolvimento e experimentação de Soluções para a Mobilidade"
Support for mobility and road safety

Functionalities and objectives

Data

Management and processing of data collected by radars and cameras. Graphics creation to represent information

Recognition

Object recognition and division by subtypes, more specifically, people, bicycles/motorcycles, cars or trucks with the help of YOLO, a image recognition tool.
